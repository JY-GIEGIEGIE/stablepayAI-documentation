## Add payment to your skill

Open your skill's `skill.md`. Paste this at the top or bottom — replace two values.

```markdown
## 💰 StablePay Payment

This Skill requires {PRICE} USDC to unlock.

Payment endpoint:
https://api.stablepay.co/pay?skill={SKILL_DID}&price={PRICE}

Verify purchase:
https://api.stablepay.co/verify?skill={SKILL_DID}&agent={AGENT_DID}
```

| Variable | Replace with |
|----------|--------------|
| `{PRICE}` | Your price in USDC (e.g., `5`) |
| `{SKILL_DID}` | Your developer DID (`did:solana:AbCdEf...`) |
| `{AGENT_DID}` | **Leave as-is** — the buyer's agent fills this at runtime |

<Tip>
  Don't have a DID yet? The section below walks you through one-time wallet setup. You only do this once — after that, adding payment to each skill is just the template above.
</Tip>

### How it works

```
Buyer's Agent reads your skill.md
    → Sees the payment block
    → Runs stablepay_pay_via_gateway in conversation
    → Signs the SPL transfer with their wallet
    → USDC sent directly to your wallet (skill_did)
    → Buyer's Agent gets access
```

### For backend-enforced security

The skill.md template is the simplest path. If you want server-side enforcement (so users can't remove the payment block), your backend should return HTTP 402 and call the verify API. See [Integrating x402 payments](/integrating-payment) for backend examples in Go, Node.js, and the x402 format reference.

### Create your wallet and DID

After configuring, restart the gateway and enter TUI:

```bash
openclaw gateway restart
openclaw tui
```

Walk through these three steps in conversation:

> Run stablepay_create_local_wallet with user_type "developer", runtime "auto"

> Run stablepay_register_local_did with user_type "developer"

> Run stablepay_configure_payment_limits with single_purchase_limit_usdc 1000, auto_purchase_threshold_usdc 50

Record the `did` from the output — that's your `{SKILL_DID}`.

### You're done

Now go back to the top of this page, copy the payment template, paste it into your skill.md, and replace `{SKILL_DID}` with your DID and `{PRICE}` with your price. That's it.
