## Before you start

You need:
- OpenClaw installed (`openclaw --version` shows ≥ 2026.3.1)
- A shell where you can set environment variables

You do **not** need:
- Docker, Kubernetes, or any backend services (the gateway at `ai.wenfu.cn` is already running)
- A Solana wallet with real funds (we use DevNet for development)
- Any prior blockchain knowledge

<Info>
  **Gateway is online.** The plugin talks to `https://ai.wenfu.cn` automatically. You don't need to start any services yourself.
</Info>

## Step 1: Install the plugin

```bash
openclaw plugins install clawhub:stablepay-agentpay-dev
```

Verify it's loaded:

```bash
openclaw plugins list
```

Expected output includes `stablepay-agentpay-dev` with status `enabled`.

## Step 2: Configure OpenClaw

Add this to your `~/.openclaw/openclaw.json`:

```json
{
  "plugins": {
    "entries": {
      "stablepay-agentpay-dev": {
        "enabled": true,
        "config": {
          "backendBaseUrl": "https://ai.wenfu.cn",
          "owsRuntime": "auto",
          "didRegisterPath": "/api/v1/did/register",
          "requestTimeoutMs": 90000
        }
      }
    }
  },
  "tools": {
    "allow": [
      "stablepay_runtime_status",
      "stablepay_create_local_wallet",
      "stablepay_bind_existing_wallet",
      "stablepay_register_local_did",
      "stablepay_configure_payment_limits",
      "stablepay_sign_message",
      "stablepay_pay_via_gateway",
      "stablepay_query_balance",
      "stablepay_query_sales"
    ]
  }
}
```

## Step 3: Set environment variables

```bash
# Required — master key for encrypting your local wallet state
export STABLEPAY_PLUGIN_MASTER_KEY="generate-a-long-random-secret-here"

# Required for real payments — DevNet hotwallet public key
export STABLEPAY_FEE_PAYER_SOL="FMNs7xqezz4bYYioPyfqPzxLLmZyJhjSzbGApMdnrC2Z"

# Optional — for unattended OWS signing
export STABLEPAY_OWS_PASSPHRASE="your-passphrase"
```

<Tip>
  Add these to your `~/.zshrc` or `~/.bashrc` so they persist across sessions.
</Tip>

## Step 4: Restart and enter TUI

```bash
openclaw gateway restart
openclaw tui
```

## Step 5: Your first conversation

In the TUI, walk through these steps in order:

### 5.1 Check runtime status

> Run stablepay_runtime_status

Confirm the output shows `has_wallet: false` and an active driver is selected.

### 5.2 Create a wallet

> Run stablepay_create_local_wallet with user_id "alice", runtime "auto"

This creates a new Solana wallet via OWS. Record the `did` and `wallet_address` shown in the output.

### 5.3 Register your DID

> Run stablepay_register_local_did with user_type "agent"

This registers your public key with the StablePay backend. You now have a valid `did:solana:...` identity.

### 5.4 Set payment limits

> Run stablepay_configure_payment_limits with single_purchase_limit_usdc 100, auto_purchase_threshold_usdc 10

- **Single purchase limit**: Maximum per-transaction spend (hard cap)
- **Auto purchase threshold**: Purchases below this amount skip confirmation

### 5.5 Make a test payment

> Run stablepay_pay_via_gateway with skill_did "did:solana:<a-seller-address>", skill_name "demo-skill", price "1.00", currency "USDC"

If the payment succeeds, you'll get a `tx_id` back.

<Tip>
  For testing, use the demo hotwallet address as the skill_did target:
  `did:solana:FMNs7xqezz4bYYioPyfqPzxLLmZyJhjSzbGApMdnrC2Z`
</Tip>

### 5.6 Check your balance

> Run stablepay_query_balance with did "<your-did>"
