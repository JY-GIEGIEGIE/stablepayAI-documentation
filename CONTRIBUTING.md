# Contribute to StablePay documentation

## How to contribute

### Option 1: Edit directly on GitHub

1. Navigate to the page you want to edit
2. Click the "Edit this file" button (pencil icon)
3. Make your changes and submit a pull request

### Option 2: Local development

1. Fork and clone this repository
2. Install Mintlify CLI: `npm i -g mint`
3. Create a branch for your changes
4. Run `mint dev` in the docs directory
5. Preview at `http://localhost:3000`
6. Commit and submit a pull request

## Writing guidelines

- **Use active voice**: "Start the payment" not "The payment should be started"
- **Address the reader directly**: Use "you" throughout
- **Keep sentences concise**: One idea per sentence
- **Tool names in code font**: `stablepay_create_local_wallet`
- **Lead with the goal**: Start with what the reader wants to accomplish
- **Include examples**: Show, don't just tell — TUI prompts, JSON payloads, terminal commands
- **Test your instructions**: Follow your own steps in OpenClaw TUI to verify they work

## What to contribute

We welcome improvements to:
- Getting started guides and tutorials
- API endpoint documentation
- x402 integration examples (especially new languages)
- MoltBay publishing guides
- Troubleshooting and FAQ content
